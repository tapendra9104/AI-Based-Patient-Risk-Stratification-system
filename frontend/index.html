Project Overview

What this project does

This system demonstrates how an AI-assisted hospital workflow can transform raw patient vitals into a risk score, a triage level, explainable top factors, and operational alerts for clinicians.

Risk Scoring Explainable AI Audit Logs Benchmark Metrics
Patient Input Validation ML Inference Risk Output Alerting
AI use in this project: the backend trains a machine learning model, predicts patient risk, highlights the strongest contributing factors, and compares model quality on a real benchmark dataset.

AI Intake and Triage

Vital signs, symptoms, and risk indicators are normalized into a structured feature set so the model can classify a patient as low, medium, or high risk.

Explainable Decision Support

The dashboard does not only show a score. It also surfaces the most important factors that pushed the patient toward a high-risk outcome.

Governance and Quality

Audit logs, model benchmarking, and role-based access make the project feel closer to a real hospital operations dashboard instead of a basic prediction demo.

Technology Stack

Which system is used to do what

The dashboard is designed as a professional full-stack workflow: the frontend handles the user experience, the backend handles validation and APIs, the ML layer performs prediction, and the platform layer manages storage, security, and deployment.

System Responsibilities

End-to-end project flow

Step 1
Patient Input: the frontend form captures vitals, conditions, and clinical indicators from the user.
Step 2
Validation: the Python backend checks types, ranges, structure, and request access before processing.
Step 3
AI Prediction: the scikit-learn model calculates a patient risk score and classifies the case into low, medium, or high risk.
Step 4
Explainability: the system identifies the most influential risk factors and returns them with the prediction.
Step 5
Persistence and Governance: the assessment is stored, recent history is updated, and audit logs keep track of critical actions.
Step 6
Visualization: the dashboard refreshes charts, tables, benchmark metrics, and the latest result card for a professional presentation.
